# Agent Configuration

## Available Tools

### Web Search
Search the internet for current information. Useful for:
- Current events and news
- Research and fact-checking
- Finding documentation and tutorials

### Browser
Visit and read web pages. Useful for:
- Reading articles and documentation
- Extracting specific information from websites
- Following up on search results

### Code Execution
Run Python code in a sandbox. Useful for:
- Mathematical calculations
- Data processing and analysis
- Testing code snippets

### Remember
Save important information to long-term memory. Useful for:
- Storing user preferences
- Saving important facts
- Tracking ongoing projects

### Recall
Search and retrieve information from memory. Useful for:
- Finding previously saved information
- Referencing past conversations
- Maintaining context across sessions

## Capabilities

- Multi-turn conversations with context awareness
- Persistent memory across sessions
- Tool usage for enhanced functionality
- Personalized responses based on user profile

## Usage Notes

- The agent reads USER.md to understand who you are
- The agent follows SOUL.md for personality and behavior
- Important information is saved to MEMORY.md
- All tools are used automatically when relevant

## Customization

Edit the files in your workspace to customize your assistant:

1. **SOUL.md** - Change personality, communication style, values
2. **USER.md** - Update your information and preferences
3. **MEMORY.md** - Manually add or edit memories

Changes take effect on the next conversation.
